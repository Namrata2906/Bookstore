ABSTRACT:-
In this project, I have tried to build an E-commerce website for selling Books , which can simulate all the basic controls that an actual shopping website has.
I have used eclipse photon, java 18.0.2 , tomcat 9.0 and MySQL Ver 8.0.28 for Win64 on x86_64 (MySQL Community Server - GPL)

SOFTWARE REQUIREMENTS :-
  Operating System - Windows XP/2003 or Linux/Solaris
  User Interface - HTML, CSS
  Client-side Scripting Programming Language - JavaScript
  Web Applications IDE/Workbench - Java, JDBC, JNDI, Servlets, JSP
  Database - MySQL
  Server Deployment - Eclipse with MyEclipse Plug-in

HARDWARE REQUIREMENTS :-
 Processor - Pentium IV
 Hard Disk - 40GB
 RAM - 256MB


INTRODUCTION :-
The "E-Commerce website" has been developed to override the problems 
prevailing in the practicing manual system. This software is supported to 
eliminate and in some cases reduce the hardships faced by this existing system. 
Moreover this system is designed for the particular need of the company to carry 
out operations in a smooth and effective manner.

Installation
To run this project, you'll need to have Eclipse IDE and Tomcat Server installed on your computer. Then, follow these steps:
1.	Clone this repository to your local machine.
2.	Open Eclipse IDE and import the project.
3.	Configure the JDBC Connector in the project by adding the JDBC driver to the project's class path and specifying the database connection details in the ConnectionProvider  in project package in java src file.
4.	Start the Tomcat Server.
5.	Run the project on the Tomcat Server.
